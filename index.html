Listas

Una lista es una generalización tanto de un vector atómico como de un data.frame. Los elementos de un vector atómico pueden ser de cualquier tipo. Supongamos que tenemos un lista dada por

  • ¿Qué sucede si a la lista x le aplico dim(x) <- c(2,3)?

El comando plot me permite graficar dos vectores de igual longtitud. Por otro lado el comando write(path) permite escribir un archivo en la dirección dada por path, por ejemplo:

{r eval=FALSE} x <-rnorm(200) write(x,file = "nicholas.txt", sep ="\n") Crea un archivo llamado archivo.txt en el directorio de trabajo. De igual forma R permite graficar en otro dispositivo. ```{r eval=FALSE} pdf(“nicholas.pdf”) # Creamos el PDF plot(rnorm(30),type = “l”) # Graficamos en el dispositivo dev.off() # Finalmente cerramos el PDF

for (i in 1:40) {
